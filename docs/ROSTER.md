# ROSTER

Each team's roster is **19 players** at all times.

## Position Minimums

Minimum players by position that you must draft and maintain during the entire season:

- 2 each **Quarterbacks**
- 6 each **Wide Receivers &/or Tight Ends**
- 5 each **Running Backs**
- 2 each **Kickers**
- 2 each **Defense**

The minimums total 17, leaving 2 roster slots that may be filled with any position.
