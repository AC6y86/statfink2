# PFL Weekly Recaps

Persona-style weekly recaps of Peninsula Football League matchups, generated by
the `/recap` slash command (see `.claude/commands/recap.md`).

## Structure

```
recaps/
  README.md                  this file
  {season}/                  one folder per season
    INDEX.md                 week-by-week table of recaps (derived from filenames)
    styles-ledger.md         persona styles already used (derived from filenames)
    storylines.md            living season-arc memory, updated each /recap run
    {season}-week{NN}-{style-slug}-style.md
    {season}-full-season-*.md  season retrospectives
    data/
      week{NN}-digest.md     the week's fact digest (sole data source for recaps)
      week{NN}-storylines.md snapshot of storylines.md as of that week
      week{NN}-factcheck.md  fact-check report for the week's recaps
      {season}-season-data.md season-wide reference data
```

## Naming convention

`{season}-week{NN}-{style-slug}-style.md` — week is always zero-padded so
lexical sort is chronological. Style slugs are lowercase, hyphen-separated.

## Workflow

`/recap <season> <week>` → generates the data digest (gated on weekly
validation), updates `storylines.md`, proposes 10 unused persona styles for
approval, spawns 10 narrator agents that write from the digest ONLY, then
fact-checks every recap against the digest. INDEX.md and styles-ledger.md are
regenerated from filenames each run.
